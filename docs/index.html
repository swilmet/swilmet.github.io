About Me

I'm a GNOME developer living in Belgium, Europe. My area of expertise revolves around text editors and other development tools using the GTK graphical toolkit.

I develop both applications and shared libraries. I'm familiar with GTK programming since 2009, and I'm a member of the GNOME Foundation since 2012.

Products

Applications

  • Maintainer of gedit, a general-purpose text editor.
  • Creator of Enter TeX, an Integrated TeX/LaTeX Environment.
  • Former maintainer of Devhelp, a developer tool for browsing and searching API documentation.

Shared Libraries

  • Co-maintainer of GtkSourceView, a source code editing widget.
  • Creator of Gedit Technology, a set of components behind gedit.
  • Creator of gspell, used for spell-checking.

Freelance Services

If you have a GTK application project and want to expand your developer team, we can work together!

If you depend on a shared library that I maintain, I can put up the work to make sure that the library provides the features that you need, and that it is a reliable piece of software.
