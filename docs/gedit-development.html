About this activity

I mainly develop the gedit and Enter TeX text editors, plus other related pieces of software.

It uses the GTK graphical toolkit, and integrates well with the GNOME desktop environment.

I develop both applications and shared libraries. I'm familiar with GTK programming since 2009, and I'm a member of the GNOME Foundation since 2012.

Main Products

Applications

  • Maintainer of gedit, a general-purpose text editor.
  • Creator of Enter TeX, an Integrated TeX/LaTeX Environment.
  • Maintainer of Devhelp, a developer tool for browsing and searching API documentation.

Shared Libraries

  • Creator of Gedit Technology, a set of components behind gedit.
  • Creator of gspell, used for spell-checking.

Syllabus

  • Author of The GLib/GTK Development Platform - A Getting Started Guide.
